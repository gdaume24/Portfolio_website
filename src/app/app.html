Geoffroy Daumer

Je suis un développeur web Java / Angular passionné par mon métier. Je suis arrivé dans le monde du développement en 2022, avec une formation "développeur en IA" par l'école Simplon x Isen. Après avoir validé cette formation, j'ai décidé de m'orienter vers quelque chose de plus général, pour avoir de bonnes bases de programmation et avoir de bons débouchés : La formation développeur full-stack Java / Angular par l'école Openclassrooms. Je suis arrivé au terme de cette formation, à force de travail acharné, c'est une vraie fierté. Je m'occupe maintenant à parfaire mon savoir et à rendre mon profil attractif, pour trouver un emploi. Mon rêve est de m'investir profondément dans la musique : création, guitare, piano, synthé, chant, et pour cela, je souhaite gagner ma vie grâce à ce domaine que j'aime qu'est l'informatique. , au cours de laquelle j'ai appris à entraîner et à mettre en production des algorithmes d'intelligence artificielle, j'ai également beaucoup fait de veille technique dans cette formation, nous avons également abordé les bases de données, le cloud, les API, tout cela en Python. J'ai eu la joie de trouver mon premier client pendant cette période, auquel j'ai été challengé pour réaliser une application qui upload rapidement des musiques sur un site de vente de musique, ce fut une fierté personnelle. À l'issue de cette formation, j'ai enchaîné avec une formation de développeur full-stack Java / Angular, où j'ai appris à développer les applications web. Cette formation a été très challengeante pour moi, j'ai appris à coder en Java et en Javascript, je maîtrise maintenant Java Spring Boot avec Maven pour le côté back, et Angular pour le côté front. Cette formation m'a également formé sur les tests (unitaires, d'intégration, end to end).
